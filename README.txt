Lets talk about the project here.

Here, I copied the template for front end from mdbootstrap ecommerce. Then, from the udemy-course: model required for eCommerce website were analyzed and created i.e item, order and order item. Then the views for cart, order summary, checkout, product details, sorting of items as per their category i.e shirt, shirtwear and outwear is done. The login and signup of user is provided. This is full djeCom web-app system.

The requirement for this project to run is given in deploy branch.
